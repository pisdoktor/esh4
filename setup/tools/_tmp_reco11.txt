11. **e-Nabız / e-Reçete entegrasyonu**
    - İzlem → USBS bildirimi (yasal zorunluluklar netleşince)

12. **Çok bölgeli SaaS veya federasyon**
    - 4.0 multi-tenant altyapısı buna uygun; merkezi güncelleme + kurum bazlı ayar

    - Mevcut yapıyı bozmadan API + hafif Vue/React sadece dashboard/plan ekranında
